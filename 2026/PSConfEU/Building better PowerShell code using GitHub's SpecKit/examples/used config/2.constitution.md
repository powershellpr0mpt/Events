/speckit-constitution
This project is a PowerShell module named PSExcuse. Apply these non-negotiable principles:
- Public functions use approved PowerShell verbs (Get-Verb) with PascalCase nouns.
- Every public function has comment-based help: synopsis, parameter descriptions, at least one example.
- Every public function is covered by Pester v5 tests. Tests must be deterministic and run with no network, no external module dependencies, and no reliance on machine state.
- Code passes PSScriptAnalyzer on default rules with no warnings.
- No external module dependencies; target PowerShell 7+.
- Any randomness is seedable/injectable so behaviour can be tested deterministically.
- Functions that write to disk support -WhatIf and -Confirm via SupportsShouldProcess.

# Building better PowerShell code using GitHub's Spec Kit

Working with AI often feels like managing a forgetful intern—you repeat instructions, fix the same mistakes, and hope it improves. The good news is, there’s a way to prevent this.

GitHub’s Spec Kit introduces Spec-Driven Development (SDD), a structured approach that gives AI the context it needs to produce consistent, high-quality PowerShell code.

In this demo-heavy session, you’ll learn how to install Spec Kit, use its features, and apply it to real-world PowerShell development. We’ll also share lessons learnt from practical scenarios so you can move from managing a forgetful intern to working with an AI that feels like a senior developer on your team.

# Experts Live NL - 2025 - Azure Monitor Baseline Alerting, taming the dragon with minimal effort

The Azure Monitor Baseline Alerting (AMBA) Framework is an amazing tool provided by Microsoft to set up Azure Monitoring for your environment.
It uses Azure Policy templates to ensure your Azure services are performing optimally.
It's based on best practices and comes with various patterns and scenarios to implement it.

In this session we'll dive into the AMBA Framework, its patterns & scenarios and how, with some extra PowerShell magic, you can keep your Azure environment in control!

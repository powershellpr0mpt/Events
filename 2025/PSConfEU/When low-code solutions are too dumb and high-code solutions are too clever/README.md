# PSConfEU 2025 - When low-code solutions are too dumb and high-code solutions are too clever

In the world of low-code and high-code solutions, sometimes you are wishing for a middle-code option.
But is there such a option? Yes! The new Logic App integration for PowerShell!

This session looks at classical Azure Logic Apps (low-code) and how to use them to create automation with a nice graphical interface, easy tracking of events and low barrier of entry.
So people with no coding skills can still follow what is going on in the system.
We will look at limitations of the classical Logic App approach and compare it to Azure Functions (high-code), where we get exponentially more power, but lose all the UI niceness.

Then we will introduce the new Logic App integration for PowerShell which will keep all the UI niceness, but also gives you all the power that PowerShell has to offer.
You will learn how to create powerfull Logic App automations, optimize them and still make your manager understand what's going on.

# PSConfEU 2023 - Diving Deeper in the MS Graph API with PowerShell - solving problems

Automating with Microsoft's Graph API can be tricky at times.
In this demo-filled session we'll go through some real life struggles while working with the Microsoft Graph API and how to solve them using PowerShell.
The items we'll go over are:

* Certificate based authentication and why you want to use it
* Throttling and what to do about it
* Using Pester to test your API calls
* How to deal with delayed output/reports

# PSConfEU 2023 - An Introduction to Microsoft Graph API and PowerShell

Microsoft's Graph API is the best thing since sliced bread!
But what is it? And more importantly, how can we use it leveraging PowerShell?
In this demo-filled session we'll try and answer these questions and more, such as:

* MS Graph API 101 (what are endpoints, methods and tokens and how do we find/use them)
* Talking to the MS Graph API using PowerShell (Invoke-WebRequest vs Invoke-RestMethod)
* Using Fiddler to find out what your doing in the Portal
* Manually getting data from MS Graph API
* How to automate it using an Azure AD Application Registration
